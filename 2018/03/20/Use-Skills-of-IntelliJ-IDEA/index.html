Use Skills of IntelliJ IDEA

Veröffentlicht am 2018-03-20
Help->Find Action(Ctrl + Shift + A)

跳转

  1. 项目之间的跳转 Ctrl+Alt+[/]
  2. 文件之间的跳转
    • 最近文件 Ctrl+E
    • 最近修改的文件 Ctrl+Shift+E
    • 上次编辑的文件 Ctrl+Shift+BackSpace
    • 上次浏览的地方 Ctrl+Alt+左箭头/右箭头
  3. 利用书签进行跳转Bookmark
    • 添加/取消书签 F11
    • 显示书签 Shift+F11
    • 添加有标记的书签 Ctrl+F11
    • 跳转到标记书签 Ctrl+数字
  4. 添加收藏add to favourite Alt+Shift+F
  5. 插件emacsIDEAs Ctrl+J
  6. 文件区和编辑区的跳转
    • 跳转到文件区 Alt+1
    • 跳转到编辑区 Esc
  7. vim

搜索

  • 类 Ctrl+N(按两次N可包含jar包中的类)
  • 文件 Shift+Ctrl+N
  • 符号(函数) Ctrl+Shift+Alt+N
  • 在路径中搜索 Ctrl+Shift+F
  • 移动光标到下/上一个单词 Ctrl+右箭头/左箭头
  • move caret to line end/start

列操作

  • 大小写切换 Ctrl+Shift+U
  • 多列操作 edit->find->Select All Occurrences(Ctrl+Alt+Shift+J)

live templates

$VAR1$
$VAR2$
$END$

postfix

fori
sout
field
return
nn
var

Alt+Enter

  • 自动创建函数
  • list replace(for转化为foreach)
  • 字符串formate或者build
  • 定义接口(接口名上按会辅助创建实现类)
  • 单词拼写(波浪线上按)
  • 导包

重构

重构函数名Shift+F6
重构方法:Ctrl+F6(或者直接Alt+Enter, 更快)

抽取

  • 场景:多个相同的字符串被函数调用
  • 抽取为本地变量 Ctrl + Alt + V
  • 抽取为静态变量 Ctrl + Alt + C
  • 抽取为成员变量 Ctrl + Alt + F
  • 抽取为方法参数 Ctrl + Alt + P
  • 抽取为函数 Ctrl + Alt + M

git的集成

Annotate
移动所有改动之处previous change
撤销修改(可撤销函数，文件，文件夹) Alt+Ctrl+Z

local history(在项目不受版本控制时通过local history保存历史记录)

显示更改历史
put label相当于提交

与spring关联

  • file->project structure (Ctrl + Alt + Shift + S)
  • facets -> “+” Spring -> “+” -> 全选

与数据库的关联

  • Data Source -> “+” -> “MySQL”

调试

  • 添加断点Toggle Line Breakpoint Ctrl + F8
  • 调试运行 Shift + F9
  • 单步运行 F8
  • resume调试到下一个断点 F9
  • 查看所有断点 Ctrl + Shift + F8
  • 禁止所有断点Mute Breakpoints
  • 条件断点 Ctrl + Shift + F8
  • 表达式求值 Alt + F8
  • 运行到指定行(光标处) Alt + F9
  • setValue F2
  • debug context configuration(运行上下文，寻找最小的上下文)
  • 调试最近运行过的 Alt + Shift + F9
  • 配置运行 Edit Configurations

其他操作

  1. 文件操作
    • 新建文件
    • 复制当前文件 F5
    • 移动文件 F6
  2. 文本操作
    • 复制文件名 Ctrl + C
    • 复制完整路径名 Ctrl + Shift + C
    • 调出最近剪切板 Ctrl + Shift + V
  3. 结构图
    • 查看代码结构(File Structure) Ctrl + F12
    • 查看Maven依赖
    • 查看类的继承关系 Ctrl + Shift + Alt + U(类名上按)
    • 类的等级树(Hierarchy) Ctrl + H / Ctrl + Alt + U(类内部按)
    • 查看方法调用层次(call hierarchy) Ctrl + Alt + H
    • 查看自身被哪些函数调用 Ctrl + Alt + H(函数名上按)
